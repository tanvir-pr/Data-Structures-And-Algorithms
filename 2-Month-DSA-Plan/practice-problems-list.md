## Problem Solving Strategy

### For Each Problem:

1. **Understand (5 min)**
   - Read problem carefully
   - Identify input/output
   - Note constraints

2. **Plan (10 min)**
   - Think of approach
   - Consider edge cases
   - Estimate complexity

3. **Implement (20-30 min)**
   - Write clean code
   - Add comments for complex logic

4. **Test (10 min)**
   - Test with sample inputs
   - Test edge cases
   - Verify complexity

5. **Review (5 min)**
   - Can it be optimized?
   - Learn from editorial if stuck

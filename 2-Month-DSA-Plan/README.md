## Important Tips

1. **Consistency > Intensity:** Study 3-4 hours daily rather than cramming
2. **Code by Hand:** Type every algorithm yourself
3. **Understand, Don't Memorize:** Focus on why, not just how
4. **Practice Daily:** Minimum 3-5 problems per day
5. **Revise Weekly:** Every Sunday, revise the week's topics
6. **Track Time Complexity:** Always analyze before coding
7. **Debug Actively:** Use pen-paper for tracing

## 🎯 Topics Covered

### Data Structures
- Arrays & Strings
- Linked Lists (Singly, Doubly, Circular)
- Stack & Queue
- Trees (Binary Tree, BST, AVL)
- Heaps (Min Heap, Max Heap)
- Graphs (Adjacency List, Matrix)
- Hash Tables
- Tries
- Disjoint Set Union (DSU)

### Algorithms
- **Searching:** Linear Search, Binary Search, Ternary Search
- **Sorting:** Bubble, Selection, Insertion, Merge, Quick, Heap, Counting, Radix
- **Recursion & Backtracking:** N-Queens, Sudoku Solver
- **Graph Algorithms:** BFS, DFS, Dijkstra, Prim's, Kruskal's, Topological Sort
- **Dynamic Programming:** Fibonacci, Knapsack, LCS, LIS, Coin Change
- **Greedy Algorithms:** Activity Selection, Fractional Knapsack
- **Bit Manipulation**
- **Mathematics:** Prime Numbers, GCD/LCM, Modular Arithmetic

## 🚀 Getting Started

### For Beginners
1. Start with the [2-Month DSA Plan README](2-Month-DSA-Plan/README.md)
2. Follow the week-by-week structure
3. Use the [progress tracker](2-Month-DSA-Plan/progress-tracker.md) to monitor your learning
4. Solve problems from the [practice problems list](2-Month-DSA-Plan/practice-problems-list.md)

### Study Routine
- **Morning (1.5 hours):** Theory and implementation
- **Evening (1.5-2 hours):** Problem solving
- **Target:** 3-5 problems per day
- **Weekly:** Revise and solve mixed problems

## 💡 Tips for Success

1. **Consistency > Intensity** - Study 3-4 hours daily
2. **Code Everything Yourself** - Don't copy-paste
3. **Understand, Don't Memorize** - Focus on concepts
4. **Practice Daily** - Minimum 3-5 problems
5. **Revise Weekly** - Review what you learned
6. **Analyze Complexity** - Always think about time/space
7. **Participate in Contests** - Test your skills
